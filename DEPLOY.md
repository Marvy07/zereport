# Zereport — Deployment Checklist

## Environments
| Environment | Branch | URL |
|---|---|---|
| Production | `main` | https://zereport.com |
| Staging | `staging` | https://staging.zereport.com |
| Preview | PR branches | Auto-generated by Vercel |

## Pre-deploy checklist

### Infrastructure
- [ ] Supabase project created (prod + staging)
- [ ] Prisma migrations run against prod DB (`npx prisma migrate deploy`)
- [ ] Supabase storage buckets created: `workspace-assets`, `generated-reports`
- [ ] Supabase RLS policies reviewed

### Auth
- [ ] Clerk production instance configured
- [ ] Clerk webhook endpoint set to `https://zereport.com/api/webhooks/clerk` (if implemented)
- [ ] Redirect URLs whitelisted in Clerk dashboard

### Stripe
- [ ] Stripe live mode enabled
- [ ] Products created: Pro ($49/mo), Agency ($99/mo)
- [ ] Price IDs copied to env vars: `STRIPE_PRO_PRICE_ID`, `STRIPE_AGENCY_PRICE_ID`
- [ ] Webhook endpoint registered: `https://zereport.com/api/webhooks/stripe`
- [ ] Webhook events enabled: `checkout.session.completed`, `customer.subscription.updated`, `customer.subscription.deleted`

### Email
- [ ] Resend domain verified (zereport.com)
- [ ] SPF, DKIM, DMARC records set
- [ ] From address confirmed: `reports@zereport.com`

### Observability
- [ ] Sentry project created, DSN copied to env
- [ ] Sentry source map upload configured (`SENTRY_AUTH_TOKEN`)
- [ ] PostHog project created, keys copied to env

### Security
- [ ] `ENCRYPTION_KEY` generated: `openssl rand -hex 32`
- [ ] `CRON_SECRET` generated: `openssl rand -hex 32`
- [ ] All env vars set in Vercel dashboard (prod + staging separately)
- [ ] `.env.local` never committed (confirmed in `.gitignore`)

### DNS
- [ ] `zereport.com` → Vercel (A/CNAME records)
- [ ] `staging.zereport.com` → Vercel staging deployment
- [ ] SSL certificates active

### Final
- [ ] `npm run build` passes locally
- [ ] `npx prisma migrate status` shows all migrations applied
- [ ] Smoke test: sign up → onboarding → create client → create report → billing page
